## 🎨 Design Elements Used

### **Color Palette**
```css
Matrix Green: #00FF41
Cyber Blue: #00D9FF
Cyber Purple: #B026FF
Dark BG: #0A0E27
Dark Card: #151A2E
```

### **Effects Applied**
- ✨ **Glass Morphism**: Frosted glass effect with backdrop blur
- 🌟 **Neon Borders**: Glowing borders in green, blue, and purple
- 💫 **Animated Orbs**: Pulsing gradient orbs in background
- 📡 **Scan Lines**: Animated scanning effect overlay
- 🎈 **Float Animation**: Subtle up/down movement
- ✨ **Glow Effects**: Pulsing neon shadows on hover
- 🎭 **Smooth Transitions**: All hover effects with smooth animations

### **Typography**
- All headings use the `cyber-gradient` class (green → blue → purple)
- Body text in `text-gray-300` for readability on dark backgrounds
- White text for emphasis and headings
- Consistent font weights and sizes
